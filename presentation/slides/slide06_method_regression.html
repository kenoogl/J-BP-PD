方法 3: 回帰モデル

線形回帰モデル (C₀, c, a₂, a₁, a₀)

coeff = β₀ + β₁·I + β₂·C + β₃·I·C

基底関数: [1, I, C, I·C]

拡張線形回帰モデル (n)

n = β₀ + β₁·I + β₂·C + β₃·I·C + (β₄ + β₅·C)/I + (β₆ + β₇·C)/I²

基底関数: [1, I, C, I·C, 1/I, C/I, 1/I², C/I²]

予測機能

  • 任意の (I, C) に対して6係数を予測
  • 予測された係数から速度欠損プロファイルを再構成
  • 実測範囲: I ∈ [0.01, 0.30], C ∈ [10, 25]
