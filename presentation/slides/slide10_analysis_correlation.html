分析 2: 係数間の相関

強い相関関係 (|r| > 0.7)

  • c ↔ a₂: r = -0.987 (非常に強い負の相関)
  • a₂ ↔ a₁: r = -0.995 (非常に強い負の相関)
  • c ↔ a₁: r = 0.973 (非常に強い正の相関)
  • c ↔ a₀: r = 0.789
  • n ↔ a₂: r = 0.721
  • n ↔ a₁: r = -0.759
  • a₂ ↔ a₀: r = -0.742
  • a₁ ↔ a₀: r = 0.721

考察

速度欠損の減衰 (c, n) と後流幅の拡大 (a₂, a₁, a₀) は密接に関連

これらの相関関係は物理的な後流発達過程を反映
